# Move contracts

This template provides two EVE builder contract examples:

- `smart_gate_extension`: typed witness smart gate extension flow
- `storage_unit_extension`: storage unit extension starter

## Build

```bash
pnpm run build:smart-gate
pnpm run build:storage-unit
```

## Publish

```bash
# testnet
pnpm run publish:smart-gate:testnet

# localnet (ephemeral)
pnpm run publish:smart-gate:localnet
```
