# Frontier BaseOps Submission Copy

## One-line pitch

`A live operations control plane for EVE Frontier organizations to manage Smart Gates, Storage Units, delegates, and proof-backed event workflows.`

## Short description

Frontier BaseOps turns native EVE Frontier primitives into one readable operator workflow. It lets organizations manage gate access, run storage-side actions, and read the resulting proof back in the same console, with each digest openable in SuiScan or SuiVision.

## Medium description

Frontier BaseOps is a base operations console for EVE Frontier. Instead of leaving Smart Gates, Storage Units, delegated execution, and proof scattered across separate scripts and implicit game knowledge, it turns them into one command surface. Operators can authorize a live route, issue a route permit, run a storage workflow, and verify the resulting digest in a single timeline. The product is intentionally framed around real utility and real Frontier-native integration, not a generic admin dashboard.

## Longer submission paragraph

Frontier BaseOps is an operator-facing control plane for EVE Frontier organizations. It combines native Smart Gate access flows, Storage Unit workflows, delegated execution patterns, and digest-backed proof into one readable command console. The current MVP demonstrates a live Utopia route authorization, one live access action, one live storage action, and then reads the resulting proof back on the same screen. The project is designed to score on utility and technical implementation at the same time: it is a real product surface built directly on Frontier-native primitives, not just a contract demo and not just a visual shell.

## Category mapping copy

- `Utility`: built for real structure operations and organization workflows
- `Technical implementation`: live contract extension, live scripts, wallet integration, digest readback
- `Live Frontier integration`: directly tied to Smart Gates, Storage Units, and native ownership models

## Three key bullets

- Native gate and storage workflows in one operator console
- Live actions with digest-backed proof on the same screen
- Official Utopia world profile plus one real owner-controlled Utopia route and proof links to SuiScan/SuiVision
